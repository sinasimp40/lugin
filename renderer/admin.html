📑 MikroTik Hotspot Setup Guide

1
Enable Hotspot Server

Go to IP > Hotspot > Hotspot Setup and follow the wizard. Select the LAN interface (e.g., bridge1 or ether2).

2
Create Hotspot User Profiles

Go to IP > Hotspot > User Profiles. Create profiles like 1hour, daily, vip. Set Session Timeout and Rate Limit per profile.

Name: 1hour
Session Timeout: 01:00:00
Rate Limit: 2M/2M
3
Configure Hotspot Server Profile

Go to IP > Hotspot > Server Profiles. Set your DNS Name (the login page URL clients see). Enable HTTPS if needed.

4
Hotspot Login Files (Optional)

Default login pages are in Files > hotspot. You can customize login.html, status.html, and logout.html. This app manages users via API — no custom page required.

5
Enable API Access

Go to IP > Services. Make sure API (port 8728) or API-SSL (port 8729) is enabled. Create a dedicated admin user in System > Users with full group for this app.

6
Connect This App to Your Router

Go to Settings (gear icon on the login page). Enter your router's IP, API port, username, and password. Click Test Connection.
